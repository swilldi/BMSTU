# Негативные
## Ввод размеров матрицы
1. Ввод не числа 
2. Ввод вещественного числа
3. < 0 
4. \> 10
## Ввод элементов
5. Ввод не числа
## Выполнение
6. Одна из размерностей 0

# Позитивные
Условие: Элементы k-го столбца упорядочены по убыванию
1. Одна колонка удовлктворяет условию
2. Все колонки удовлктворяет условию
3. Несколько колонок удовлктворяет условию
4. Матрица 1хN, 1 <= N <= 10
5. Матрица 10х10
6. Матрица Nx1, 1 <= N <= 10, удовлетворяет условию
7. Матрица Nx1, 1 <= N <= 10, не удовлетворяет условию

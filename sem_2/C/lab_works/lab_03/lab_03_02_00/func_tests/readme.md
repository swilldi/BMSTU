# Позитивные
хорошая строка -- это строка в которой количество элементов, 
начинающихся на заданную цифру равно количеству элементов, 
заканчивающихся на эту же цифру

1. Все строки хорошие
2. Нет хороших строк
3. Первая строка хорошая
4. Последняя строка хорошая
5. Матрица 10х10 + хорошие строки
6. Матрица 1х1 + хорошие строки

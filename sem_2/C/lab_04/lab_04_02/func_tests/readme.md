# Негативные
1. Пустая строка
2. Введена строка > 256 символов (включая нулевой)
3. Длина слова превышает 16 символов (включая нулевой)
5. Только разделители

# Позитивные
1. Строка содержит одно слово, которое встреается несколько раз
2. Нет повторяющихся слов
3. Несколько различных слов
4. Присутствует слово из 16 символов (включая нулевой)
5. Общая длина строки 256 символов (включая нулевой)
7. Слова разделяются не пробелами, а символами пунктуации (,;:-.!?)

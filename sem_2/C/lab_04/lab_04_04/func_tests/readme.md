# Негативные
1. Пустоая строка
2. Введена строка > 256 символов

## IP корректый (т.е. из чисел из строки можно составить корректный IP)
1. Много пробелов в начале
2. Много пробелов в конце
3. Много пробелов в конце и начале
4. Разделено не точками
5. Нет рараздилителей
6. Лидирующие нули, количество не больше 2-х
7. Лидирующие нули, количество больше 2-х
8. Несколько раздилетелей
9. Оканчивается разделителем
## IP НЕкорректый
10. Много пробелов в начале
11. Много пробелов в конце
12. Много пробелов в конце и начале
13. Разделено не точками
14. Нет рараздилителей
15. Лидирующие нули, количество не больше 2-х
16. Лидирующие нули, количество больше 2-х
17. Несколько раздилетелей
18. Отрицательное число
19. Числов больше 255
20. Чисел меньше 4-х
21. Заканчивается разделителем
22. Начинатеся разделителем
23. Буквы в строке
## IP корректый (т.е. количество чисел небольше 16 и максимальное числа их трех цифр подряд 255)
24. Начинается разделителем

# Негативные
1. Пустая строка
2. Введена строка > 256 символов
3. Длина слова превышает 16 символов (включая нулевой)
4. Все слова одинаковые
5. Одно слово

Из слов, отличных от последнего, составляется новая строка. Слова в резуль-
тирующую строку помещаются в обратном порядке с одним пробелом в качестве
разделителя. После последнего слова пробел не добавляется.

Удалить из слова все последующие вхождения первой буквы

# Позитивные
1. Несколько слов отличны от последнего
2. Все слова отличны от последнего
3. Слова содержат несколько вхождений первой буквы
4. Слова не содержат вхождений первой буквы
5. Слова длиной 16 символов (включая нулевой)
6. Строка 256 символов (включая нулевой)
7. слово начинается с цифр

\hypertarget{index_intro}{}\doxysection{\texorpdfstring{Введение}{Введение}}\label{index_intro}
Это проект на языке C для реалзизации следующего алгоритма для каждой строки файла~\newline
1) Найти строке словв с масимальным и минимальным количеством гласных~\newline
2) Поменять найденые слова местами~\newline
3) Записать полученую строку в новый файл~\newline

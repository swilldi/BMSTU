# Задание
Задан файл, содержащий строки текста, состоящие из слов, отделенных друг от друга
произвольным количеством пробелов (алфавит латинский). В каждой строке найти слова с наибольшим и наименьшим количеством гласных букв (a, e, i, o, u, y) и поменять их
местами. Результат записать в новый файл.

# Допущения

1. Символы из таблицы ASCII
2. Регистр не учитывается
3. Слово может располагаться только на одной строке
4. Максимальная длина одного слова 30 символов
5. Максимальная длина строки 100 символов
6. Максимальное количество слов в строке 10
7. Подается текстовий файл
8. Если в строке нет гласных букв, то строка записывается в исходном виде
9. Если в строке содержится несколько слов с минимальным или максимальным количеством гласных, то выбираются первые встреченные в строке.
10. Если количество гласных во всех словах одинаково, то строка записывается в исходном виде
11. Пустые строки пропускаются
12. Слова разделяются пробелами
13. Под словом подразумевается любая последовательность символов из таблицы ASCII

```Pseudocode

Вход:
	words – Массив строк
	n – количество слов

	индекс слова с наибольшим количеством гласных = i_max
	индекс слова с наименьшим количеством гласных = i_min

	min = max = количество гласных в words[0]

	i = 1 – индекс слова
	
	Пока i < n

			max = count_vowel
			i_max = i
		конец если

		если count_vowel < min
			min = count_vowel
			i_min = i
		конец если
		
		i = i + 1
	Конец пока
```

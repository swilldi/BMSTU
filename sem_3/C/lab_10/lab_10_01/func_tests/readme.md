Каждый предмет имеет три характеристики
    - Название (строка)
    - Цену (целое число)
    - Вес (целое число)
В файле каждое поле предмета содержится на новой строке

insert используется для добавления предмета в заранее выбранное место
pop_back используется для удаления первого или последнего предмета, из последовательности предметов, упорядоченного по алфавиту
reverse используется, чтобы получить инвентарь предметы, упорядоченые в обратном алфавитном порядке
sort используется чтобы упорядочивать предметы по возрастанию цены

Осмысленанная задача:
Реализация инвентаря персонажа, в который можно добавлять новый предмет по указаному индексу. Есть возможность удалять предмет,
находящийся на последнем месте. При использовании совестно с сортировкой используется для удаления наиболее ценного предмета, 
например продажы предмета. При использовании совестно с сортировкой и разворотом используется для удаления наиболее дешевего предемета,
например для очистки инвентаря от дешевых предметов, чтобы уменьшить общую массу инвентаря.

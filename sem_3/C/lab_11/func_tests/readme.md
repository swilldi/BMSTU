## Тесты для %s
1. Спецификатор в начале
2. Спецификатор в конце
3. Спецификатор в середине
4. Спецификатор перед текстом (%sSomeText)
5. Спецификатор в тексте (Some%sText)
6. Спецификатор после текста (SomeText%s)
7. Несолько спецификаторов подрадя (%s%s)
8. Несколько спецификаторов разделяются (%s SomeText %s)
9. Только спецификатор
10. Для полной вставки спецификаторов хватает места
11. Для полной вставки спецификаторов требуется всё место в строке

# Негативные
## Ввод команды
1. Отрицательное число
2. Не число
3. При выборе режима работы программы число больше 2
4. При выборе типа данных на котороых основан стек число больше 2
5. При выборе действия число больше 6
6. Пустая строка
7. Добавляемая строка больше 63
8. 
# Позитивные
(Тесты в двух вариантах, для стека на статическом массиве и на списке)
1,2. Добавление одного элемента
3,4. Добавление нескольких элементов
Поправить 5,6. Довление и удаление не всех элементов
7,8. Вывод пустого стека сразу после начала
9,10. Вывод пустого стека после добавления и удаления элементов
11. Добавление 64-х элементов
